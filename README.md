# Simple CRUD Web App

A basic full-stack web application built using Node.js, Express.js, EJS, and CSS.

## Features
- Add and display data
- Edit existing entries
- Server-side rendering using EJS

## Tech Used
Node.js, Express.js, EJS, CSS, JavaScript

## Run Project
npm install  
node index.js
